# 🛍️ Nostra - Responsive E-commerce Website

Nostra is a modern and responsive fashion e-commerce website built using HTML, Tailwind CSS, and JavaScript. The project focuses on creating a clean user interface with responsive design and interactive features.

## 🚀 Features

- Responsive Home Page
- Responsive Collections Page
- Responsive Contact Page
- Mobile Navigation Menu
- Hero Image Slider
- Product Search Functionality
- Product Filter Functionality
- Contact Form Validation
- Newsletter Subscription Validation
- Responsive Product Cards
- Modern Tailwind CSS Design

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- Font Awesome
- Google Fonts
